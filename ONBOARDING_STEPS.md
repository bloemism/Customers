# Stripe Connect オンボーディング手順

## 現在の状況

- **連結アカウントID**: `acct_1SmtPlHk8MTQ5wk4`
- **タイプ**: `none`（Standardアカウントとして設定されていません）
- **決済有効**: `false`
- **詳細提出**: `false`

## 必要な情報

以下の情報を提供する必要があります：

1. **business_profile.mcc** - ビジネスのMCCコード（業種コード）
   - 例: 花屋の場合、`5999`（その他の小売店）または`5970`（工芸品・工芸用品店）
2. **business_profile.product_description** - 商品説明
   - 例: 「花屋・フラワースクール」
3. **external_account** - 銀行口座情報
   - 銀行名、支店名、口座番号、口座種別（普通/当座）
4. **tos_acceptance.date** - 規約への同意日
   - オンボーディングページで自動的に設定されます
5. **tos_acceptance.ip** - 規約への同意IPアドレス
   - オンボーディングページで自動的に設定されます

## オンボーディング手順

### ステップ1: オンボーディングリンクを作成

1. ブラウザで `http://localhost:5173/create-account-link` にアクセス
2. 連結アカウントID（`acct_1SmtPlHk8MTQ5wk4`）が既に入力されていることを確認
3. 「オンボーディングリンクを作成」ボタンをクリック
4. 作成されたリンクをコピーまたはクリック

### ステップ2: Stripeのオンボーディングページで情報を入力

1. オンボーディングリンクをクリックしてStripeのページを開く
2. **規約に同意**（必須）
3. **ビジネス情報を入力**:
   - ビジネス名
   - ビジネスタイプ（個人/法人）
   - 商品説明（例: 「花屋・フラワースクール」）
   - 業種コード（MCCコード）
4. **銀行口座情報を入力**:
   - 銀行名
   - 支店名
   - 口座番号
   - 口座種別（普通/当座）
5. **本人確認情報を入力**（必要に応じて）
6. **送信**をクリック

### ステップ3: オンボーディング完了の確認

オンボーディング完了後、以下を確認してください：

- `charges_enabled`: `true`
- `details_submitted`: `true`
- `type`: `standard`
- `restrictions`: 空

### ステップ4: 決済を試す

1. `http://localhost:5173/stripe-connect-payment` にアクセス
2. 決済金額を入力
3. 「Stripe Connectで決済する」ボタンをクリック
4. 決済が正常に処理されることを確認

## トラブルシューティング

### オンボーディングリンクが作成できない場合

- ローカルAPIサーバーが起動しているか確認（`http://localhost:3000`）
- 環境変数`STRIPE_SECRET_KEY`が正しく設定されているか確認

### オンボーディング完了後も決済が有効にならない場合

- Stripe Dashboardで連結アカウントの状態を確認
- 必要な情報がすべて提供されているか確認
- 数分待ってから再度確認（反映に時間がかかる場合があります）
